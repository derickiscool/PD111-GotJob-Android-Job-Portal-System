# 3.4 Recruiter Job Seeker Communication

## 3.4.1 Description and Priority
This feature facilitates direct, real-time communication between recruiters and potential candidates. It ensures both parties are promptly informed of interactions through a centralized notification and chat system.
**Priority:** Essential


## 3.4.2 Functional Requirements
The following functional requirements govern the communication and notification systems:

* **[SRS11]** Recruiter and job seeker shall be able to exchange messages via chat. (Essential)
* **[SRS12]** The system shall notify users when new chat messages are received. (Essential)
* **[SRS13]** User shall be able to view their recent chats and notifications from the portal. (Essential)

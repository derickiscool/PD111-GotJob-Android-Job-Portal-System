# 4.1 User Interfaces

## 4.1.1 Description
This section defines the technical constraints and delivery expectations for the client-facing front end. It ensures that the application is accessible on the target platform specified by the customer.

## 4.1.2 Interface Requirements
The following requirement captures the specific delivery constraints for the client application:

* **[SRS14]** The system shall deliver a native Android client application that provides user interfaces for job seekers and recruiters.  (Essential)
